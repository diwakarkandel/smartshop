# SmartShop Backend

SmartShop Backend is a scalable e-commerce backend application built with **Spring Boot**, **Gradle**, and **MySQL**.  
It provides REST APIs for managing products, categories, users, authentication, orders, and other core e-commerce features.

---

## Features

- User registration and login
- Authentication and authorization
- Product management
- Category management
- Cart management
- Order management
- Customer management
- RESTful API design
- MySQL database integration
- Scalable backend structure for future growth

---

## Tech Stack

- **Java 21**
- **Spring Boot**
- **Spring Data JPA**
- **Spring Security**
- **MySQL**
- **Gradle**
- **Lombok**
- **Hibernate**
